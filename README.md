# Enterprise AI Security & Productivity Copilot

Production-ready SaaS platform for enterprise AI-powered document intelligence, code security review, meeting analysis, and team productivity management.

## Tech Stack

- **Frontend**: Next.js 14+ TypeScript
- **Backend**: FastAPI Python
- **Database**: PostgreSQL
- **AI**: OpenAI/Gemini/Claude
- **Vector DB**: ChromaDB

## Quick Start

```bash
docker-compose up --build
```

## Features

- Multi-tenant workspace
- JWT authentication
- Role-based access control
- Document upload & RAG
- Code security review
- Meeting analysis
- Task management
- Security alerts
- Audit logs
